# Prayer Time Widgets

Display-only "today's prayer times" widgets for **Homey Pro**, **Android**, and
**iOS** (via Scriptable — no App Store app of my own). Shows Dawn, Sunrise, Noon,
Maghrib, Midnight for a chosen UK city, plus a static "next prayer" label.

Cities: **London, Cardiff, Glasgow, Manchester, Newcastle**. No alarms, no audio —
just a beautiful little card.

> Full run instructions per platform are added in Task E1. Data source and parsing
> are documented in [`shared/parsing-contract.md`](./shared/parsing-contract.md).
